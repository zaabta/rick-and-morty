# Rick And Morty


**How to use**:

1. Clone the template repository:

```bash
    > git clone https://github.com/zaabta/rick-and-morty
```

2. Install dependencies:

```bash
    > yarn
    or
    > npm install
```

3. Run the application:

```bash
    > yarn start
    or
    > npm start
```

4. Build the application:

```bash
    > yarn build
    or
    > npm run build
```
